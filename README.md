Sekigae

## プロジェクトについて

席替え用のランダム生成スクリプトです。  
毎回ランダムな座席配置を生成できます。

## 開発環境構築

Python 3.11 をインストールしてください。

### 実行方法

bashでプロジェクトフォルダへ移動後：

```bash
python ./main.py
```

実行するとランダムな席替え結果が表示されます。
